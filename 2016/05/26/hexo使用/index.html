hexo使用

发表于 2016-05-26   |   分类于 hexo   |     |   阅读次数 次

hexo博客搭建需要准备nodejs,以及git

首先在github上创建一个 xxx.github.io的respsitory
安装hexo
npm install hexo-cli -g
初始化项目
hexo init xxx.github.io
安装依赖
cd xxx.github.io
npm install
安装hexo管理插件，编辑发布blog
npm install hexo-admin --save
更换主题
git clone https://github.com/iissnan/hexo-theme-next themes/next
安装git部署插件，发布项目
npm install hexo-deployer-git --save

修改_config.yml 
deploy:
type: git
repository: https://github.com/xx/xx.github.io.git
branch: master

hexo deploy #发布到github上
###### 在blog中添加RSS订阅需要安装hexo-generator-feed 
npm install hexo-generator-feed --save

安装完hexo-generator-feed后，在配置文件_config.yml中添加如下配置

 plugin:
- hexo-generator-feed
feed:
type: atom
path: atom.xml
limit: 20

添加之后，运行hexo g,就会在页面上生成RSS图标。

评论系统

添加多说评论


  1. 登录多说，首页中选择”我要安装”

  2. 创建站点，填写表单。多说域名

  3. 在_config.yml中新增
    duoshuo_shortname
    字段，值为多说域名中所填的内容。
    修改

    duoshuo_info:
      ua_enable: true
      admin_enable: false
      user_id: xxx
      admin_nickname: author
    duoshuo_hotartical: true #开启热评文章

    user_id: xxx中的 xxx 为自己的id

数据统计与分析

数据统计可以使用百度统计、Google分析、腾讯分析和不蒜子统计等
本blog中使用不蒜子统计。
修改_config.yml

busuanzi_count:
  enable: true
  site_uv: true
  site_uv_header: 访客数
  site_uv_footer: 人数
  site_pv: true
  site_pv_header: 访问量
  site_pv_footer: 次
  page_pv: true
  page_pv_header: 阅读次数
  page_pv_footer: 次
内容分享

next中有百度分享、多说分享、JiaThis、AddThis等，
本blog中使用的为github中开源项目share.js。
